# Airbnb.dashboard


Global Airbnb Performance Dashboard

1. Project Title / Headline

A concise Power BI dashboard analyzing global Airbnb data to uncover trends in listings, pricing, customer behavior, and market dynamics.

⸻

2. Short Description / Purpose

This dashboard provides a clear, interactive view of Airbnb performance across major cities. It helps identify growth patterns, pricing strategies, and user behavior for better data-driven decisions.

⸻

3. Tech Stack

* Power BI Desktop – Data visualization & report building
* Power Query – Data cleaning and transformation
* DAX – Measures, KPIs, and calculations
* Data Modeling – Relationships for cross-filtering

⸻

4. Data Source

Airbnb dataset containing:

* Listings, hosts, and reviews
* Property types and pricing
* City-level ratings and trends

⸻

5. Features / Highlights

* Listings peaked in 2015, then declined (COVID impact)
* Entire Place dominates listings
* Paris, NYC, Sydney lead market share
* Hotel rooms are highest priced
* 98.8% users review only once
* 66.9% hosts fully verified (strong trust signal)
* Seasonal trends vary by region

⸻

6. Business Value

* Understand market concentration and demand
* Identify pricing inefficiencies
* Analyze customer engagement behavior
* Evaluate trust and platform reliability
